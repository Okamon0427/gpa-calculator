# Todo - App
Calculate your GPA. You can also estimate GPA for upcoming term to get target GPA in total.

---
## URL
https://gpa-calculator-14a4b.web.app/

## Getting Started
copy this project into your local machine
- install npm
```
yarn install 
```
- launch app
```
yarn start
```

## Tech Stack
- React
- TypeScript
- Ant Design
- Firebase
